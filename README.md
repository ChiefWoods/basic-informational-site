## Getting Started

### Prerequisites

Update your Bun toolkit to the latest version.

```bash
bun upgrade
```

### Setup

1. Clone the repository

```bash
git clone https://github.com/ChiefWoods/basic-informational-site.git
```

2. Install all dependencies

```bash
bun install
```

3. Start server

```bash
bun run start
```
